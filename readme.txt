Programas necesarios para ejecutar back-end y front-end:
	* Python3 (corriendo en consola)
	* Java (corriendo en consola)

Instrucciones para correr back-end:
1.- Desde la consola ir a la carpeta out/artifacts/
2.- Ejecutar el siguiente comando:
java -jar ProyectoIntegrador_jar.jar

Instrucciones para correr front-end:
1.- Desde consola ir a carpeta raiz del proyecto
2.- Ejecutar el siguiente comando:
python3 client.py
